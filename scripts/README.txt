================================================================================
                        Text-to-Speech 文本转语音工具
                           Windows GUI 版使用文档
================================================================================

一、部署说明
--------------------------------------------------------------------------------
    将压缩包解压到任意目录即可使用，无需安装 Python 或其他开发环境。

    目录结构:
    ├── 文本语音自动化测试工具.exe   # GUI 主程序
    ├── tts.exe                      # 命令行工具
    ├── config.json                  # 配置文件
    ├── text.txt                     # 文本文件 (每行一条)
    ├── generate.bat                 # 批量生成语音 (命令行)
    ├── play.bat                     # 播放模式 (命令行)
    ├── ttsengine/                   # 离线TTS引擎 (已包含)
    ├── models/                      # 离线语音模型 (已包含)
    ├── ffmpeg/                      # 音频处理工具 (已包含)
    ├── adb/                         # ADB工具 (已包含)
    └── output/                      # 输出目录 (wav/log/png)

    【注意】使用 Piper 离线语音时，首次运行需安装 VC++ 运行库:
    下载地址: https://aka.ms/vs/17/release/vc_redist.x64.exe


二、快速开始
--------------------------------------------------------------------------------

    1. 双击运行 "文本语音自动化测试工具.exe"

    2. 在"生成语音"标签页:
       - 点击"打开文件编辑"修改 text.txt
       - 点击"刷新列表"查看文本
       - 点击"批量生成语音"开始生成

    3. 在"播放模式"标签页:
       - 输入设备 IP 并点击"连接设备"
       - 或点击"检查连接"验证 USB 连接
       - 点击"开始播放"运行测试

    4. 在"配置设置"标签页:
       - 选择语音引擎和声音
       - 可视化编辑语音模板
       - 点击"保存配置"应用更改


三、语音引擎说明
--------------------------------------------------------------------------------

    本工具支持两种语音引擎:

    1. Edge TTS (在线，推荐)
       - 需要联网
       - 音质更好，支持多种声音
       - 可选声音:
         * 云健 (播音男声) [默认]
         * 云希 (标准男声)
         * 云扬 (新闻男声)
         * 云夏 (少年男声)
         * 晓晓 (标准女声)
         * 晓伊 (甜美女声)
         * 晓北 (东北女声)
         * 晓妮 (陕西女声)

    2. Piper (离线)
       - 无需联网
       - 需要安装 VC++ 运行库
       - 声音: 中文女声


四、语音模板编辑
--------------------------------------------------------------------------------

    在"配置设置"标签页可以可视化编辑语音模板:

    - 点击 "+ 添加语音" 添加语音片段
    - 点击 "+ 添加静音" 添加静音片段
    - 使用 ↑↓ 按钮调整顺序
    - 点击"删除"移除片段
    - $MAIN 代表主文本内容，不可删除

    默认模板结构:
    [静音1秒] → [小翼管家] → [静音2秒] → [文本内容] → [静音18秒]
             → [小翼管家] → [静音2秒] → [返回首页] → [静音5秒]


五、播放模式说明
--------------------------------------------------------------------------------

    播放模式需要连接 Android 设备，用于:
    1. 播放语音文件
    2. 记录设备 logcat 日志
    3. 在语音结束前自动截图

    连接方式:
    - USB: 开启设备的 USB 调试，连接电脑后点击"检查连接"
    - 无线: 输入设备 IP 地址，点击"连接设备"

    测试结果保存在 output/ 目录:
    - *.wav  语音文件
    - *.log  logcat 日志
    - *.png  截图
    - test_report.txt  测试报告


六、命令行工具
--------------------------------------------------------------------------------

    除了 GUI，还提供命令行工具 tts.exe:

    # 单条生成
    tts.exe -text "你好世界" -output hello.wav

    # 批量生成（使用配置文件）
    tts.exe

    # 批量生成（简单模式，无前后缀）
    tts.exe -file text.txt -output output -simple

    # 播放模式
    tts.exe -play

    # 播放指定目录
    tts.exe -play -playdir output/场景1

    # 重新生成默认配置
    tts.exe -init


七、常见问题
--------------------------------------------------------------------------------

    Q: Edge TTS 连接失败
    A: 检查网络连接是否正常，确保能访问外网

    Q: piper 执行失败 (exit status 0xc0000409)
    A: 缺少 Visual C++ 运行库。请下载安装:
       https://aka.ms/vs/17/release/vc_redist.x64.exe

    Q: ADB 连接失败
    A: 确保设备已开启 USB 调试，或检查 IP 地址是否正确

    Q: 生成的语音有问题
    A: 检查 text.txt 编码是否为 UTF-8

    Q: 界面显示异常
    A: 确保系统已安装 WebView2 运行时 (Windows 10/11 通常已内置)
